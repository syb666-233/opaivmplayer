LunaTrickcal v1.0.0 正式版 — 版本说明
======================================

发布标识：正式版（Stable Release）
发布日期：2026-07-06
适用平台：Windows 64 位

本版本包含
----------
✓ LunaTranslator 定制运行时（OCR 韩文 → 简中翻译）
✓ 游戏内黄色译文浮层（可拖动，右键删除）
✓ 预置 Trickcal UI 词典 39 条
✓ 国服中文文本池 12 万+ 条（译后纠错）
✓ 运行时自动学习词典（方案 C，无需 Frida）
✓ 游戏专属 / 全局专有名词翻译配置
✓ 一键导出学习词典

不包含（后续版本再考虑）
------------------------
✗ 全窗口自动 OCR（已搁置，性能与体验待优化）
✗ Frida 抓词工具（仅维护者开发用，不随正式版分发）
